# Secure Password Generator

> A Python-based tool to generate strong and randomized passwords using multiple character sets.

## Features
- Generates secure passwords with customizable length
- Uses uppercase, lowercase, digits, and special characters
- Input validation for minimum password length
- Option to generate multiple passwords

## Tech Stack
- Python
- random module
- string module

---

## How It Works
The program combines multiple character sets and randomly selects characters to generate a strong password of the desired length.
